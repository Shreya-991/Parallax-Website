# 🌄 Basic Parallax Website

A simple and visually appealing **Parallax Scrolling Website** built using **HTML and CSS**. This project demonstrates how to create depth and motion effects using layered images and CSS properties.

---

## 🚀 Features

* 🌌 Smooth **parallax scrolling effect**
* 🎯 Layered **background and foreground animation**
* 🏞️ Three adventure sections:

  * Biking 🚴‍♂️
  * Paragliding 🪂
  * Surfing 🏄‍♂️
* 💡 Clean and beginner-friendly code
* 📱 Responsive layout (basic)

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* Google Fonts (Poppins)

## 🎨 How It Works

* The **parallax effect** is achieved using:

  * `perspective` on the parent container
  * `translateZ()` and `scale()` for depth
* Background images use:

  * `background-attachment: fixed` for scrolling illusion
* Foreground and background layers move at different speeds to create a **3D effect**

## ▶️ How to Run the Project

1. Download or clone the repository
2. Make sure all image files are in the correct folder
3. Open `index.html` in your browser
